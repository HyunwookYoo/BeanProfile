판단 ①

이름 · 로스터리 자동추측 안 함

어떤 줄이든 이름일 수 있어 오탐 위험 → 칩으로 배정하거나 직접 입력. 국가·날짜·가공·로스팅단계·컵노트만 자동.

판단 ②

Copy-on-save

촬영본은 임시 경로로 두고 저장할 때만 앱 보관함으로 복사 → 취소 시 고아 파일 없음.

판단 ③

칩 → 텍스트 필드만

칩은 문자열이라 이름·로스터리·지역·농장·품종·고도·메모·컵노트에만 배정. 날짜·enum은 피커로.

확인 요청

상세 사진 = 썸네일 vs 배너

스펙시트 정체성을 위해 옆 썸네일을 제안(E). 풀폭 배너를 원하면 바꿉니다.
